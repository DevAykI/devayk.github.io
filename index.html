Projects

WhatsApp Vault Bot

Node.jswhatsapp-web.jsSQLitesharp

Staff across five stores spent 10 to 15 minutes per order scrolling through WhatsApp group chats to find photos. Built an event-driven bot that monitors group chats, indexes images to order numbers in real time, and emails them on demand. Pitched on a Monday, deployed across all five stores by Wednesday.

Command Reference
Full command reference. The bot supports 20+ commands for fetching, searching, indexing, diagnostics, and performance reporting across all store locations.
Fetch Command
A staff member retrieves order photos with a single command. The bot finds all indexed images, emails them, and confirms delivery with file count and source store.
Email Delivery
The email received after a /fetch command. Photos are compressed, labelled by order number, and chunked into multiple emails if they exceed Gmail's 25MB limit.
Cross-Store Stats
Index summary across all store locations. The bot tracks thousands of photos and orders per store, with cached file counts and coverage dates.

Multi-Store Retail Automation System

Node.jsPlaywrightGoogle Sheets APITelegram Bot API

Manual stock counting and returns investigation consumed roughly an hour per store per day. Built a polling-based automation system that handles stock counts, RMA processing, and serial number investigation via concurrent browser automation, reducing per-item investigation time by over 90%.

Auto Investigate
Triggered by a checkbox in the RMA sheet. Reads a serial number and fault reason, investigates on the portal, classifies as RMA / RTO / Unlikely Stock, and writes results back to the sheet.
Auto RMA Fill
Downloads and processes the weekly RMA report. Investigates serial numbers to determine buy order, tester, price, and box ID, then updates the RMA spreadsheet.
RMA Output
Formatted RMA result with investigation details, tester name, and auto-generated QR code formula for box labelling.
Hub Page
Remote trigger using a Google Sheets checkbox and dropdown selection with live status updates. Operators select a store and click Run.
